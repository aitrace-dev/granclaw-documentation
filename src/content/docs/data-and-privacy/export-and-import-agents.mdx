Agent card → export button → zip downloads. The zip contains everything:

- Vault (all markdown files)
- Conversation history
- Browser profile (saved cookies and logins)
- Agent configuration
- Task board

## Importing

Dashboard → **Import** → drop the zip → confirm. The agent appears in your list in seconds, no manual setup.

The imported agent is identical to the original: same memory, same browser logins, same schedules, same task board.

<Callout type="tip">
Export before wiping. The wipe cannot be undone — the export is your restore path.
</Callout>
