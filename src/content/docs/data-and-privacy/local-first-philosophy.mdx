> *"Tools should not own the craftsman."*

GranClaw runs on your machine. Everything is stored locally. Nothing leaves your hardware except the LLM calls you make to the provider you chose.

No account. No sign-up. No telemetry. No phone home.

## Bring your own LLM

OpenAI, Anthropic, Gemini, Groq, OpenRouter. Configure per agent, swap anytime, mix providers across agents. No provider is the default. No lock-in.

## Durable formats

| Data | Format |
|---|---|
| Agent memory | Plain markdown in [`vault/`](/agent-superpowers/vault-obsidian-memory) |
| Conversation history | SQLite |
| Configuration | Plain JSON |
| [Secrets](/data-and-privacy/secrets) | Encrypted, env vars only |

If GranClaw is abandoned tomorrow, every file is still readable. Open the vault in any text editor. Query history in any SQLite browser. Edit config in any editor.
