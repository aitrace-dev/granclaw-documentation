## Live view

While the agent has an active browser session, open the live view panel. You get a real-time stream of the agent's screen. Opening the panel doesn't interrupt the agent — it keeps working.

## Replay

When a session ends, the panel becomes a replay. A WebM recording of the full session with chapter markers for every command the agent ran — jump directly to any action without scrubbing.

<Callout type="note">
Recordings are stored locally inside the agent's workspace. Never sent anywhere.
</Callout>

The live and replay views show what's on screen. For reasoning and tool calls, see the chat panel and logs view.
