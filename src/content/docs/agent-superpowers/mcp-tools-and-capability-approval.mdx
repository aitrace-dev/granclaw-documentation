MCP tools extend what your agent can do — Linear, GitHub, email, databases, anything with an MCP integration. Install from agent settings → **Tools**. Restart the agent to pick it up.

Tools are scoped per agent. Installing for one agent doesn't affect others.

## Capability approval (three tiers)

| Tier | What happens |
|---|---|
| **Auto-approve** | Low-risk, matches your allow list. Granted immediately. |
| **Auto-deny** | On your block list (shell access, destructive ops). Denied always. |
| **Ask user** | Falls outside both lists. Surfaced in your dashboard — you approve or deny. |

Think of it as sudo for your agent. The agent can request elevated access; it cannot grant itself access.

<Callout type="note">
Capability approval is a Guardian feature, coming in a future release. Until then, access is based on the agent's allowed tools configuration.
</Callout>

## Revoking

Agent settings → **Tools** → uninstall. Takes effect on next restart. Individual capability approvals can be revoked from the guardian's approval history view.
