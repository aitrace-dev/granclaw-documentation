Same experience as the dashboard — from your phone. Instant acknowledgment, typing indicator, live status board as each tool runs.

Replies are sent in the language you write in (English, Spanish, Chinese — auto-detected).

## Setup

1. Message **@BotFather** on Telegram → `/newbot` → follow prompts → copy your bot token.
2. Agent settings → **Telegram** → enable → paste token → save.
3. Open Telegram, find your bot, send a message.

<Callout type="tip">
The token is stored encrypted in the secrets vault. To rotate it, paste a new one and save.
</Callout>

## What works over Telegram

Everything in the dashboard chat: browsing tasks, vault queries, workflow status, instructions. Long replies are split automatically at Telegram's 4000-character limit.

## Dashboard-only

- Live browser stream
- Agent settings and schedule configuration
- File uploads
