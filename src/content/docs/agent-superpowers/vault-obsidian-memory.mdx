Every agent has a vault — a folder of plain markdown files where it keeps everything it wants to remember. Open it in [Obsidian](https://obsidian.md) and you're browsing your agent's memory like any other notebook.

## What the vault contains

- Daily journals, action logs, topic notes, research findings
- `[[wikilinks]]` between notes — a real graph the agent navigates for context

The agent writes to the vault on its own. No configuration required.

## Opening in Obsidian

Default path: `~/.granclaw/workspaces/<agent-id>/vault/`

Obsidian → **Open folder as vault** → navigate there. All features work: graph view, backlinks, canvas, search. The wikilinks the agent wrote resolve as native Obsidian links.

## Writing to the vault yourself

Add, edit, or delete files like any other markdown. The agent reads your notes on its next turn — useful for giving it project context, preferences, or corrections.

## Portability

The vault is just files. It persists across restarts and is included in every agent export. Plain text that outlasts any software.
