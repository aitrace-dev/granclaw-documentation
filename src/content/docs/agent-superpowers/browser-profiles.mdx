Each agent has its own persistent Chrome profile. Log in to any site once and the agent reuses that session every time — no API key needed, no re-authentication.

## First login

Ask the agent to open a browser and navigate to the site. It lands on the login page, completes authentication, and saves the session. Done.

<Callout type="tip">
For 2FA sites: use the [live browser panel](/agent-superpowers/browse-control) to complete the 2FA step yourself the first time. The profile saves the authenticated cookie — future sessions skip 2FA unless it expires.
</Callout>

## Isolation

Each agent reads from its own profile directory. No two agents share cookies or saved logins — ever.

## Storage

Profiles live in `~/.granclaw/workspaces/<agent-id>/`. They're included in [agent exports](/data-and-privacy/export-and-import-agents) and travel with the agent to a new machine.

## Expired sessions

When a session expires the agent lands on a login page. Walk it through login again — the new session is saved for future turns.
