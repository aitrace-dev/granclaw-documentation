Open the dashboard → Settings (top-right icon). Click **Add provider**, pick one, paste your key, save.

![Provider Settings page with model and API key configuration](/images/settings.png)

## Supported providers

| Provider | Models | Keys |
|---|---|---|
| Anthropic | Claude Sonnet, Haiku, Opus | [console.anthropic.com](https://console.anthropic.com/) |
| OpenAI | GPT-4o and others | [platform.openai.com](https://platform.openai.com/api-keys) |
| Google Gemini | Pro, Flash | [aistudio.google.com](https://aistudio.google.com/app/apikey) |
| Groq | Fast open models | [console.groq.com](https://console.groq.com/keys) |
| OpenRouter | Routes to dozens of models | [openrouter.ai](https://openrouter.ai/keys) |

You only need one to start. Each agent can use a different provider — swap anytime from agent settings, no restart needed.

<Callout type="tip">
Your key is stored locally and injected as an environment variable only at the moment a message is sent. It never touches version control.
</Callout>
