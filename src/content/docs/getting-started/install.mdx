## Requirements

- **Node 20+** — check with `node --version`
- **macOS or Linux**
- **An API key** from one of: OpenAI, Anthropic, Gemini, Groq, or OpenRouter

## Install

```
npx granclaw@beta
```

Starts the backend at [http://localhost:3001](http://localhost:3001) and the dashboard at [http://localhost:5173](http://localhost:5173). Press `Ctrl+C` to stop.

## Verify

Open [http://localhost:5173](http://localhost:5173) — you should see the GranClaw dashboard.

<Callout type="note">
A provider error on first load is expected — you haven't added an API key yet.
</Callout>
