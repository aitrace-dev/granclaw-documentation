## Schedules

Agent settings → **Schedules** → **Add schedule**.

- **Cron expression** — standard five-field syntax. `0 8 * * 1-5` = every weekday at 8am. The UI shows a plain-language preview as you type.
- **Prompt** — exactly what you'd type in chat: "Summarize the news from my RSS feeds and post the top three to my vault."

Each run gets its own conversation channel. Click any run in the timeline to read its full output.

## Workflows

Saved multi-step pipelines: agent calls, code blocks, and LLM calls chained together. Define once, invoke by name, combine with a schedule.

Steps can reference the output of previous steps via template variables.

Create workflows in agent settings → **Workflows** tab.
