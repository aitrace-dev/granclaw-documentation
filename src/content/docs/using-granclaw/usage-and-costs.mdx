Sidebar → **Usage**. Organized by agent, session, and date.

## What's tracked per session

| Field | What it is |
|---|---|
| Input tokens | Prompt sent to the model (history + tool results) |
| Output tokens | Model's reply |
| Cache reads | Tokens served from prompt cache (fraction of the cost) |
| Cache writes | Tokens written to cache (small premium, pays off on reuse) |
| Model | Which model handled this session |

## Cost estimates

Shown next to token counts, based on provider pricing at release time. These are estimates — your actual bill comes from your provider.

<Callout type="note">
OpenRouter users: costs reflect OpenRouter's rates, not upstream pricing. Check OpenRouter's usage dashboard for your actual charges.
</Callout>

To find expensive outliers, sort the session table by token count descending.
