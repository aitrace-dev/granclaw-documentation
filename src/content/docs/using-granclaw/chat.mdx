Click an agent card → type → Enter. Replies stream token by token. Tool calls appear inline as they run — expand any one to see the full input and output.

## Sidebar panels

- **[Mission Control](/using-granclaw/mission-control)** — the agent's kanban. Ask it to plan something and watch cards appear.
- **Logs** — recent tool calls for the current session, without scrolling through chat.

## Stop, history, danger zone

- **Stop** — interrupts the current action cleanly. You can send a new message immediately.
- **History** — scroll up to read past sessions. All stored locally, always available.
- **Danger zone** — wipes conversation history, [vault](/agent-superpowers/vault-obsidian-memory), and databases. Irreversible.

<Callout type="warning">
[Export the agent](/data-and-privacy/export-and-import-agents) before wiping. The wipe cannot be undone.
