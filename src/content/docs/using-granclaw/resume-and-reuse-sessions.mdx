Close the browser, shut down your machine, restart GranClaw — your agent picks up exactly where you left off.

## What persists

- Conversation history — every message, in order
- [Vault](/agent-superpowers/vault-obsidian-memory) — all markdown notes the agent has written
- Browser logins — cookies and saved sessions in the agent's [Chrome profile](/agent-superpowers/browser-profiles)
- [Mission Control](/using-granclaw/mission-control) — every kanban card in its current state
- [Schedules](/using-granclaw/schedules-and-workflows) — fire on time even when you're not looking

The only thing that doesn't survive is an open live browser stream — reopen the panel to reconnect.

## Browser session reuse

Each agent has its own Chrome profile. Log in to LinkedIn, Gmail, or any site once and the agent reuses that session forever — no CAPTCHA loops, no re-authentication.

<Callout type="tip">
Profiles are completely isolated. One agent's logins are invisible to every other agent.
</Callout>

## Backup and portability

[Export](/data-and-privacy/export-and-import-agents) any agent to a zip: vault, history, browser profile, config, task board — everything. Import on another machine and the same agent comes back online, memory intact.
