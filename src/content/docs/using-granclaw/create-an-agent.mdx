Click **+ New Agent** in the dashboard, fill in three fields, save. The card appears immediately.

## The three fields

- **Name** — lowercase slug: `research-agent`, `linkedin-writer`. Becomes the agent's ID.
- **Model** — from your configured providers: `claude-sonnet-4-5`, `gpt-4o`, `gemini-2.0-flash`.
- **Tools** — what the agent can do. Default covers filesystem and browser.

Everything else (history, [vault](/agent-superpowers/vault-obsidian-memory), [task board](/using-granclaw/mission-control), [schedules](/using-granclaw/schedules-and-workflows)) is created automatically on first message.

<Callout type="tip">
Each agent is fully isolated — its own workspace, [browser profile](/agent-superpowers/browser-profiles), and [vault](/agent-superpowers/vault-obsidian-memory). Run as many as you need in parallel.
</Callout>

## Custom workspace

Agent settings → **Advanced** → set an absolute path. Useful when you want the agent working inside a project folder you already have open.
